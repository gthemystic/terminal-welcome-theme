### ğŸ”§ Manual Installation

If you prefer to install manually:

1. **Copy the modules:**
```bash
mkdir -p ~/.zshrc.d
cp terminal-welcome-theme/.zshrc.d/*.zsh ~/.zshrc.d/
```

2. **Update your `.zshrc`:**
```bash
# Add this to ~/.zshrc (or use the provided .zshrc.example)
export ZSH_MOD_DIR="${ZSH_MOD_DIR:-$HOME/.zshrc.d}"
typeset -g ZSHRC_START_TIME=$(($(date +%s%N)/1000000))
for module in "$ZSH_MOD_DIR"/[0-9][0-9]-*.zsh(N); do
  [[ -f "$module" ]] && source "$module"
done
```

3. **Restart your shell:**
```bash
exec zsh
```

### Add Custom ASCII Art

Add a new case statement in the `_get_cute_ascii()` function:

```bash
11) # Your custom character
  echo "\033[38;5;215m        Your ASCII art here\033[0m"
  echo "\033[38;5;215m        ğŸ¨ Your message! ğŸ¨\033[0m"
```


### Add Custom Images

The welcome screen supports displaying random images! Just add `.png`, `.jpg`, `.gif`, or `.webp` files:

```bash
mkdir -p ~/.config/terminal-welcome/
# Copy your images here
cp ~/Pictures/cool-image.png ~/.config/terminal-welcome/
```

The system will randomly show images 80% of the time if any exist.

### Disable the Welcome Screen

To disable the welcome screen temporarily:
```bash
unset ZSHRC_WELCOME_SHOWN
```

Or set it before opening the shell to skip this session:
```bash
ZSHRC_WELCOME_SHOWN=1 zsh
```

## ğŸ” How It Works

### Modular System

All files in `~/.zshrc.d/` are automatically loaded in numerical order (00-99):
- `00-init.zsh` â†’ Foundation (PATH, timing)
- `70-welcome.zsh` â†’ Welcome screen (loaded after init)

Add your own modules with any number (e.g., `50-my-aliases.zsh`).

### Caching Strategy

To keep startup fast, the welcome screen caches:
- **System stats** (10-minute TTL) - Memory, disk, uptime
- **Git activity** (5-minute TTL) - Recent commits across repos

Cache files:
- `/tmp/zsh-system-stats-cache`
- `/tmp/zsh-git-activity-cache`

Clear cache manually:
```bash
rm /tmp/zsh-*-cache
```

### Performance

Typical startup time:
- **First run:** ~500-800ms (depends on git repos)
- **Cached runs:** ~200-400ms
- **With images:** ~400-600ms

## ğŸ› ï¸ Troubleshooting

### Welcome screen doesn't show

1. Check that `00-init.zsh` and `70-welcome.zsh` are in `~/.zshrc.d/`
2. Verify your `.zshrc` includes the module loading code
3. Restart shell: `exec zsh`

### Git activity shows nothing

Make sure you have git repos in `~/Developer/`:
```bash
mkdir -p ~/Developer
# Clone/create some repos there
```

### Images don't display

1. Install `viu` (image viewer for terminals):
   ```bash
   brew install viu  # macOS
   cargo install viu  # Alternative
   ```
2. Add images to `~/.config/terminal-welcome/`

### Slow startup

Clear caches:
```bash
rm /tmp/zsh-*-cache
```

Or check what modules are loaded:
```bash
typeset -f | grep "^[a-z]" | wc -l  # Count functions
```
